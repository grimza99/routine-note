# iOS MVP 출시 플랜 (KR TestFlight 2주 + 6주 일정)

## 1) 목표

- 공개 출시일: `2026-03-24 (화)`
- 초기 4주 목표: 설치 수 최대화 + 치명 이슈 0건
- KPI 우선순위
  1. 설치 수
  2. D1 유지율
  3. 크래시프리 세션 비율

## 2) 범위/기본값

- 타깃 국가: 대한민국 (iOS 우선)
- 제품 범위
  - Native: 로그인, 운동기록, 루틴
  - WebView fallback: 리포트, 챌린지, 마이페이지
- 데이터 수집
  - 내부 이벤트 수집(`/api/events`) 기준 대시보드 운영
  - 외부 MMP 도입은 차기 단계

## 3) 6주 실행 일정

### Week 1-2 (`2026-02-18 ~ 2026-03-02`) - 클로즈드 TestFlight

- 대상: 50~150명
- 배포: 주 2회 (화/금)
- 수집 이벤트:
- `app_install`
- `app_open`,
- `login_success`,
- `workout_created`,
- `workout_updated`,
- `workout_removed`,
- `routine_applied`,
- `report_viewed`,
- `workout_sets_created`,
- `workout_sets_updated`,

- 통과 기준
  - 로그인 → 운동 저장 성공률 95% 이상
  - 치명 크래시 0
  - 상위 불편 이슈 Top 10 수집 완료

### Week 3 (`2026-03-03 ~ 2026-03-09`) - 안정화 스프린트

- 상위 이슈 수정 + 재현 케이스 문서화
- 회귀 테스트 수행
- App Store 메타데이터 초안 완료
  - 앱 설명/키워드/스크린샷/개인정보 라벨
- 운영 SLA 확정: 문의 24시간 내 1차 응답

### Week 4 (`2026-03-10 ~ 2026-03-16`) - 심사 제출 준비

- RC 빌드 1개 고정
- 릴리즈 후보 체크리스트 100% 충족
- 심사용 계정/데모 시나리오/리뷰 노트 작성
- 제출 목표일: `2026-03-13 (금)`

### Week 5 (`2026-03-17 ~ 2026-03-23`) - 심사 대응

- 리젝 대응 SLA: 24시간 내 수정/재제출
- 사전 마케팅 노출 3회(인스타/커뮤니티)

### Week 6 (`2026-03-24 ~ 2026-03-30`) - 공개 출시 주간

- 공개 출시: `2026-03-24 (화)`
- 콘텐츠: 릴스 3종 고정 포맷 + 설치 CTA
- 운영: 리뷰/문의 일일 모니터링, 설치 대시보드 1일 2회 점검

## 4) Go/No-Go 게이트

1. 치명 버그(P0) 0건
2. 크래시프리 세션 99.5% 이상
3. 로그인→운동 저장 퍼널 완료율 95% 이상
4. 이벤트 누락률 2% 미만
5. 심사 리젝 고위험 항목 점검 완료
   - 권한 고지
   - 개인정보 라벨
   - 계정 삭제 정책

## 5) API/계약 고정 항목

1. `POST /api/events` 스키마 확장
   - 필드: `sessionId`, `screenName`, `funnelStep`, `errorCode?`
2. 모바일 메타 헤더 강제
   - `x-client-platform`, `x-app-version`, `x-app-build`
   - 누락 시 서버 경고 로그
3. 딥링크/라우팅 계약 동결
   - 계약 버전: `2026-03-10-v1`
   - 동결일: `2026-03-10`

## 6) 테스트 시나리오

### 기능 E2E

- 회원가입/로그인/토큰 갱신
- 운동 저장(세트 포함), 루틴 적용, 재진입 후 데이터 일관성

### 출시 리스크

- 오프라인/저속망
- 토큰 만료/401 재시도
- 앱 백그라운드 복귀
- 딥링크 진입/WebView 왕복

### 데이터 정확성

- 설치→오픈→로그인→운동저장 이벤트 시퀀스 검증
- 앱 버전별 이벤트 집계 일치성 검증

### 심사 대응

- 개인정보/계정관리/문의 경로 노출 확인
- 심사용 데모 계정 플로우 3분 완주 확인

## 7) 운영/마케팅 실행

1. 채널
   - 메인: 인스타 릴스
   - 보조: 운동 커뮤니티 1개
2. 콘텐츠
   - 주 3회: 10초 기록 데모, 루틴 적용 전/후, 월간 리포트 요약
3. 응대
   - 리뷰/문의 24시간 내 1차 응답
   - 주 1회 VOC 리포트
4. 개선 루프
   - 설치 증가 포맷 반복
   - 저성과 포맷 즉시 중단
   - VOC 기반 개선 백로그 전환

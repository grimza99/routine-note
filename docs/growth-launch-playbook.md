# 클로즈드 테스트 기반 출시/홍보 플레이북

## 1) 목표

- 출시 전 사용자 피드백을 확보해 공개 출시 품질과 평점을 올린다.
- 초기 핵심 KPI는 `설치 수`, 보조 KPI는 `D7 재방문율`로 본다.

## 2) 타깃

- 국내 중급 이상 운동 사용자
- 루틴 관리/기록 지속성이 필요한 사용자

## 3) 채널

- 메인: 인스타/릴스
- 보조: 운동 커뮤니티 1개(디스코드/오픈채팅 중 택1)

## 4) 콘텐츠 운영

- 주 3회 업로드
  - 기록 10초 데모
  - 루틴 적용 전/후
  - 월간 리포트 요약
- CTA 문구 고정
  - 설치 링크
  - 한 줄 가치 제안

## 5) 단계별 운영

### 사전 2주 (클로즈드 테스트)

- 테스터 모집 폼 공개
- 주당 피드백 정리 1회
- 저평점/불편 포인트 우선 수정

### 출시 주간

- 릴스 3종 고정 포맷 업로드
- FAQ/문의 응답 SLA 운영 (24시간 내 응답)

### 출시 후 4주

- 상위 성과 포맷 반복
- 저성과 포맷 폐기
- 리뷰 코멘트 기반 개선 항목 백로그화

## 6) 추적 이벤트

- `app_install`
- `app_open`
- `login_success`
- `workout_saved`
- `routine_applied`
- `report_viewed`

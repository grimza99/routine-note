# routine-note-mobile

하이브리드 RN 앱 워크스페이스입니다.

## 실행

1. 루트에서 `pnpm install`
2. `pnpm mobile:start`

## 환경 변수

- `EXPO_PUBLIC_API_BASE_URL` (예: `http://127.0.0.1:3000`)
- `EXPO_PUBLIC_APP_VERSION` (예: `0.1.0`)
- `EXPO_PUBLIC_APP_BUILD` (예: `1`)

## 1차 범위

- Native: 로그인, 운동기록, 루틴
- WebView fallback: 리포트, 챌린지, 마이페이지
